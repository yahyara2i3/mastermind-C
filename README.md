# 🎯 Jeu Mastermind en C

Ce projet a été réalisé dans le cadre de ma première année en Génie Informatique à l'EST (École Supérieure de Technologie). Il s'agit d'une implémentation du célèbre jeu de société Mastermind en langage C, jouable directement dans la console.

## 🚀 Fonctionnalités
* Génération aléatoire d'une combinaison secrète.
* Saisie et vérification des propositions du joueur.
* Affichage du nombre de pions bien placés et mal placés.
* Gestion du nombre d'essais (victoire ou défaite).

## 🛠️ Prérequis
Pour compiler et jouer, vous avez besoin d'un compilateur C comme GCC installé sur votre machine (Windows, Linux ou macOS).

## 💻 Comment compiler et jouer ?

1. Clonez ce dépôt sur votre machine :
`git clone https://github.com/yahyara2i3/mastermind-c.git`

2. Compilez le fichier source via le terminal :
`gcc main.c -o mastermind` *(Remplacez main.c par le nom de votre fichier si différent)*

3. Lancez le jeu :
* Sous Linux/Mac : `./mastermind`
* Sous Windows : `mastermind.exe`
